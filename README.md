# **Capstone Project: Off To Taiwan**
## React Native Front End

### Tools/Packages Used
- React Navigation
- Firebase
- Moment.js
- React Native FBSDK
- React Native Google Sign In
- React Native Maps
- React Native Splash Screen
- React Native Config
- Native Base
- Axios

### Get Started
- npm install
- cd ios, pod install
- npm start
- npm run ios
- build app in xcode
